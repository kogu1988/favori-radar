# Chrome Web Store Yayın Dosyaları

Bu klasör Chrome Web Store formuna girilecek metinleri ve inceleme notlarını
içerir.

## Yayın Sırası

1. `npm run release` komutunu çalıştırın.
2. Chrome Web Store Developer Dashboard'da yeni öğe oluşturun.
3. `release/favori-radar-1.0.0.zip` dosyasını yükleyin.
4. `LISTING.md` içindeki mağaza metinlerini kullanın.
5. Gizlilik politikası URL'si olarak
   `https://kogu1988.github.io/favori-radar/` adresini forma girin.
6. `REVIEW-INSTRUCTIONS.md` içeriğini test talimatlarına ekleyin.
7. Ekran görüntülerini ve 128x128 mağaza ikonunu yükleyin.
8. Dağıtımı ilk incelemede `Unlisted` seçerek test edin; onay sonrası isterseniz
   `Public` yapın.

## Kalan Manuel İşler

- Gizlilik politikasını herkese açık bir HTTPS sayfasında yayınlamak.
- Gerçek verileri maskeleyerek en az bir 1280x800 veya 640x400 ekran görüntüsü
  hazırlamak.
- Geliştirici hesabı iletişim e-postasını doğrulamak.
- Sahibinden.com ile resmi bağlantı bulunmadığını mağaza açıklamasında korumak.
